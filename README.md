# QucPanel Package

Public distribution assets for the QucPanel one-click installer.

Install:

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/Y3y202/QucPanel-package/master/install.sh)"
```

This repository publishes:

- `install.sh`
- `qucpanel-dist.tar.gz`
- `qucpanel-linux-amd64.tar.gz`
- `SHA256SUMS`
